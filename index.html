HTML

  • The head element contains information about the webpage or metadata.
  • The body element represents the visible content shown to the user.
  • Child elements are nested inside other elements known as parent elements.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS rules can be applied to text and images
  • CSS defines the colors, aesthetics, visual layut, and organization
  • Inline, Internal, and External are three methods of adding CSS to your HTML file.
  • External sheets is the most popular.
  • A selector defines the element or attributes to which the rules or declarations will apply.
  • Declarations contain 2 components:CSS Property and Value Property.
  • Link elements creats the connection to the external file.
  • rel attribute specifies the relationship between the current document and the linked document resource.
  • href attribute specifies the location (URL) of the external resource.
  • box-shadow property adds shadow effects around an elements frame.
  • Using the box-shadow method you can set multiple effects separated by commas.
  • box-shadow property is described by x and y offsets relative to the element, blur, spread radius, and color.
  • The X offset defines the width or extent of the shadow on the x-axis (or to the right of the box).
  • The Y offset defines the extent on the y-axis (or just below the box).
  • we style elements using the class attribute .card

GIT

  • git status checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches.
  • git add -A will stage your changes
  • git commit -m "" is important especially when collaborating with other fellow developers.
  • git pull origin main command is to ensure once changes are added and commited locally, your local branch is up to date before pushing up your changes.
  • git push origin "" command will push up changes to the remote repository.

JavaScript

  • High level dynamic programming language that allows users to interact with websites.
  • One of its core features is variables, which store data types as values.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements allow us to interrupt the control flow.
  • An array is a single variable that is used to hold a group of data that is related in some way.
  • Arrays can store different types of data (string, number, Boolean values).
  • In JavaScript arrays are zero-indexed and sequential.
  • Loops use a predictable pattern of indices to perform a task on all items in a array.
  • Length is a built in property of an array.
  • A declaration uses a var keyword to declare a variable.
  • Functions are used to control when the code executes.
  • Functions are building blocks of JS.
  • Functions group a set of instructions that tells a computer how to perform a task.
  • Functions must be defined and called
  • math.floor() and math.random() are both properties of JavaScript's built in math object that can be used together to generate a random number.
